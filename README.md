# 忘れ物チェックAI

登録した教材画像と、その日に撮った確認写真をGeminiで照合し、
確認済みを黄緑、未確認を赤で表示するStreamlit試作アプリです。

## できること

- 教材名と画像を登録
- 「指定した日だけ」「毎週」「毎日」を設定
- 指定日の持ち物を自動表示
- カメラ撮影または写真アップロード
- Geminiによる登録画像との照合
- 黄緑・赤のカード表示
- AIの誤判定を手動修正
- SQLiteによるローカル保存

## 1. Pythonを準備

Python 3.11以上を推奨します。

## 2. 必要なライブラリを入れる

このフォルダをターミナルで開き、次を実行します。

```bash
pip install -r requirements.txt
```

## 3. Gemini APIキーを設定

`.streamlit/secrets.toml.example` をコピーして、
ファイル名を `.streamlit/secrets.toml` に変更します。

中身の

```toml
GEMINI_API_KEY = "ここにAPIキー"
```

へGoogle AI Studioで取得したAPIキーを入れます。

APIキーは他人に見せたり、GitHubへ公開したりしないでください。

## 4. アプリを起動

```bash
streamlit run app.py
```

ブラウザが自動で開きます。

## 使用順序

1. 「持ち物を登録」で教材名・画像・必要日を登録
2. 「今日・指定日の確認」で日付を選ぶ
3. 持ち物を机に並べて撮影
4. 「AIで持ち物を確認する」を押す
5. 黄緑・赤の判定結果を確認
6. 誤判定は手動ボタンで直す

## 注意

この版は探究・試作用です。似たノート、裏返った教材、重なった物、
暗い画像などでは誤判定することがあります。

より完成度を上げる場合は、教材1個につき正面・裏面・背表紙など
複数の登録画像を保存する仕組みや、OCRによる表紙文字の照合を
追加すると改善できます。
